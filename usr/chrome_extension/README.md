# English to Arabic Converter Extension

This is a Chrome Extension that converts selected English text into Arabic characters using the `Alt+R` shortcut.

## Installation

1. Open Google Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked**.
4. Select the `chrome_extension` folder from this project.

## Usage

1. Navigate to any website (e.g., Google, WhatsApp Web, Facebook).
2. Type some English text in an input field or text area.
3. Select the text you want to convert.
4. Press `Alt + R`.
5. The text will be converted to Arabic based on the configured mapping.

## Supported Elements
- Standard `<input>` fields
- `<textarea>` elements
- `contenteditable` elements (Rich text editors, chat inputs)
